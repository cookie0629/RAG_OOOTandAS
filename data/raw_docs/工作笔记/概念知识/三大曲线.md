从工业软件底层内核的角度来看，从贝塞尔到 B 样条，再到 NURBS，是一个**为了满足工业制造需求而不断进化、不断打补丁直至完美**的过程。

### 1. 贝塞尔曲线 (Bézier Curve)

- **一句话概括：** 通过一组控制点“整体牵引”生成的平滑曲线，**牵一发而动全身**。
    
- **工业软件视角：**
    
    - **优势：** 算法极简，极其稳定。它是早期工业设计（如雷诺汽车）的功臣，数学表达非常直观。
        
    - **劣势（致命伤）：** 1. **全局控制：** 你拖动其中任意一个控制点，整条曲线的形状都会跟着变。这在复杂的工业产品设计中是灾难，设计师无法进行局部微调。
        
        2. **无法表达圆：** 无论你用多少阶的贝塞尔曲线，都无法完美画出一个标准的正圆，只能无限逼近。
        
    - **连续性 (G0-G2)：** 想要将两条贝塞尔曲线拼接达到 G1（相切）或 G2（曲率连续），需要极度苛刻地对齐连接处的控制点，拼接难度极大。
        

### 2. B 样条曲线 (B-Spline)

- **一句话概括：** 贝塞尔曲线的“分段打补丁”升级版，实现了**局部控制**，动一个点只影响它附近的一小段。
    
- **工业软件视角：**
    
    - **优势：** 引入了“节点向量 (Knot Vector)”的概念，把一条长曲线切成了很多段。设计师拖动一个控制点，只有相邻的几段曲线会变形，其他部分纹丝不动。并且在较低阶数下（如 3 阶），就能自然获得非常优秀的 G2 连续性。
        
    - **劣势：** 虽然解决了局部控制和拼接连续性的问题，但它**依然无法精确表达圆、椭圆等解析几何图形**。
        
    - **连续性 (G0-G2)：** B 样条在系统内部是分段的，但它在节点处的连接非常顺滑，工业软件通常利用 3 阶（Cubic）B 样条来轻松实现整条曲线的 G2 连续。
        

### 3. NURBS 曲线 (Non-Uniform Rational B-Spline，非均匀有理 B 样条)

- **一句话概括：** 工业界**终极大一统**的几何表达标准，引入了“权重”，既能自由捏造复杂曲面，又能绝对精确地表达标准的圆和圆锥曲线。
    
- **工业软件视角：**
    
    - **优势：** NURBS 是当今所有顶级工业软件（CATIA, Siemens NX, SolidWorks）的底层标准。这里的“有理 (Rational)”意味着它给每个控制点增加了一个**权重 (Weight)** 属性。通过调整权重，NURBS 不仅保留了 B 样条的所有优点（局部控制），还能将工业中极其重要的标准几何体（直线、圆柱、圆锥、球面）完美统一到一个数学模型中。
        
    - **劣势：** 数据结构庞大，数学计算极其复杂。当你对包含大量 NURBS 曲面的模型进行布尔运算时，非常考验几何内核的稳定性和计算能力。
        
    - **连续性 (G0-G2)：** 工业界用来做“A级曲面（Class-A Surface，如汽车外壳）”的终极武器。它能极其精准地控制接缝处的 G2（曲率连续，保证斑马线测试平滑）甚至 G3（曲率变化率连续）级别的过渡。

### 核心对比总结表

| **维度**      | **贝塞尔曲线 (Bézier)** | **B 样条 (B-Spline)** | **NURBS**             |
| ----------- | ------------------ | ------------------- | --------------------- |
| **控制方式**    | **全局控制** (动一点变全身)  | **局部控制** (动一点变局部)   | **局部控制 + 权重控制**       |
| **能否精确表达圆** | 不能，只能逼近            | 不能，只能逼近             |  **完美精确表达**           |
| **多段拼接难度**  | 极高 (很难保持 G2)       | 较低 (内部原生平滑)         | 最低 (工业界拼接 A 面标配)      |
| **工业应用场景**  | 字体设计、简单矢量绘图 (SVG)  | 复杂但不要求绝对圆度的自由形态     | **航空航天、汽车外壳、高精度模具设计** |
| **数学复杂度**   | 简单                 | 中等                  | **极其复杂** (包含节点和权重)    |

### 1. G0 连续（位置连续 / Position）

- **图像表现：** 两根线条的端点确确实实连在一起了，中间没有断开，但是连接处有一个**明显的尖角或折点**。
    
- **开车体验（撞墙急转弯）：** 假设你开着一辆车沿着轨迹走，到了 G0 连接点，你需要瞬间把方向盘打死，车子会产生极其剧烈的顿挫，甚至感觉像撞了一下。
    
- **触觉感受：** 摸过去扎手。
    

### 2. G1 连续（切线连续 / Tangent）

- **图像表现：** 尖角被磨平了。在连接点处，两根曲线的“走向”是完全一致的，看起来是一条平滑的线。经典的例子就是操场的跑道（两头是半圆，中间是直线，直线和圆弧的连接处就是 G1）。
    
- **开车体验（猛打方向盘）：** 车子在直道上开，突然进入固定弯道。虽然轨迹是连贯的，但在进入弯道的一瞬间，你需要**瞬间把方向盘从回正状态打到一个固定角度**。这时候车子虽然没撞墙，但乘客会感受到一股突然出现的离心力，身体会被猛地甩向一边。
    
- **光影表现：** 摸起来不扎手了，但在工业设计中，如果用高光照射，反光带（高光点）在这个连接处会发生**扭曲或者明显的折叠**。
    

### 3. G2 连续（曲率连续 / Curvature）

- **图像表现：** 不仅连接处的走向一致，连“弯曲的剧烈程度”（曲率）也是完全一致且平滑过渡的。
    
- **开车体验（丝滑入弯）：** 高速公路或高铁轨道的弯道设计。在进入弯道前，有一段“缓和曲线”。你的方向盘是**逐渐、均匀地打过去的**。离心力也是慢慢增加的，乘客杯子里的水都不会洒出来。
    
- **光影表现（工业界的“斑马线测试”）：**
    
    这是工业软件中检验曲面质量最常用的方法（Zebra Analysis）。
    
    - 如果是 **G1** 曲面拼在一起，斑马线能连上，但在接缝处会有一个**尖锐的拐角**。
        
    - 如果是 **G2** 曲面拼在一起，斑马线不仅连得上，而且过渡得**如丝般顺滑，没有任何折痕**。这就是为什么苹果手机的外壳、高级汽车（如保时捷）的流线型车身，都强制要求使用 G2 甚至更高级别的曲面过渡，因为只有这样，环境光打在产品表面时，才不会有廉价的“塑料折光感”。
